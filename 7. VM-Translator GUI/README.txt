Es funktioniert so:
Öffnet man die Jar-Datei so wird man aufgefordert eine Vm-Datei auszusuchen. Danach wird eine GUI geöffnet die einem 2 Textfelder anzeigt:
Im linken steht der Text aus der VM Datei und im rechten der entstandene Text für die Asm Datei.
Unter dem Rechten Textfeld ist ein Button womit man eine neue ASM-Datei erzeugt, die im selben Verzeichnis ist wie die ausführbare Jar-Datei und denselben Namen hat wie die ausgesuchte VM-Datei.
Wenn man eine weitere VM-Datei übersetzen möchte, so muss man das Programm schließen und erneut öffnen.
